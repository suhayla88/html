Computer Science and Web Development Terminology

HTML (HyperText Markup Language)
A markup language used to create the structure of web pages.
CSS (Cascading Style Sheets)
A style sheet language used to describe the presentation of a web page's HTML or XML content.
JavaScript
A high-level programming language used to create interactive effects within web browsers.
API (Application Programming Interface)
A set of protocols and tools for building software and applications. It defines the methods and data structures for interaction between software components.
DOM (Document Object Model)
An interface that represents the structure of a document (e.g., HTML or XML) as a tree of nodes, allowing programs to manipulate the document's structure and style.
Frontend
The client-side part of a web application that users interact with directly, often using HTML, CSS, and JavaScript.
Backend
The server-side part of a web application that handles data processing, databases, and server-side logic.
Responsive Design
A design approach that ensures web pages render well on various devices and screen sizes by using flexible layouts, images, and media queries.
Framework
A pre-built collection of tools, libraries, and best practices used to streamline development, like React, Angular, or Django.
SEO (Search Engine Optimization)
The process of optimizing web pages to rank higher in search engine results, improving visibility and traffic.
URL (Uniform Resource Locator)
The address used to access a resource on the internet, such as a web page or file.
HTTP (HyperText Transfer Protocol)
The protocol used for transferring hypertext requests and data on the web.
HTTPS (HyperText Transfer Protocol Secure)
A secure version of HTTP that encrypts data between the client and the server using SSL/TLS.
Web Hosting
The service that allows individuals and organizations to make their website accessible via the World Wide Web.
Database
A structured collection of data that is stored and accessed electronically, typically used to store information for web applications.
AJAX (Asynchronous JavaScript and XML)
A technique used in web development for creating asynchronous web applications, allowing data to be loaded in the background without refreshing the page.
Git
A distributed version control system used for tracking changes in source code during software development.
Cloud Computing
The delivery of computing services like storage, processing, and networking over the internet, often using remote servers.
Version Control
A system that tracks changes to a file or set of files over time, allowing developers to collaborate and manage different versions of code.
Cross-Browser Compatibility
The ability of a website or web application to function correctly across different web browsers, ensuring a consistent experience for all users.
